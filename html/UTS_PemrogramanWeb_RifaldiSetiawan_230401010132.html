HTML (HyperText Markup Language) adalah bahasa standar yang digunakan untuk membuat dan menyusun halaman web. HTML memberikan kerangka dasar dari sebuah halaman web dan memungkinkan pengembang web untuk menentukan struktur dan tata letak konten, contoh elemen dasar pada HTML adalah <head>, <title>, <body>, apabila kita ingin menulis elemen html di dalam html, maka kita harus escape tag seperti ini &lt;p&gt; (Bisa dilihat dalam kodingan mengenai cara saya menapilkan head/title/body)

CSS, atau Cascading Style Sheets, adalah bahasa stylesheet yang digunakan untuk menambahkan style ke elemen-elemen HTML di sebuah halaman web. Dengan CSS, kita dapat mengontrol tampilan dan tata letak dari berbagai elemen HTML dengan cara yang lebih fleksibel dan efisien. Contoh penerapan CSS dalam stylesheet mencakup berbagai aspek desain, seperti mengubah warna tulisan, menambahkan margin dan padding, menentukan jenis dan ukuran font, mengatur tata letak menggunakan grid atau flexbox, dan banyak lagi. Dengan menggunakan CSS, kita bisa memastikan bahwa halaman web tampil konsisten di berbagai perangkat dan ukuran layar, serta meningkatkan estetika dan keterbacaan konten web secara keseluruhan.
